## INSTALLATION [Termux] :
* `pkg update`
* `pkg install -y git wget`
* `git clone https://github.com/rohitswami876604/apktool-in-termux.git`
* `mv apktool_2.7.0.jar apktool.jar`
* `chmod +x apktool.jar`
* `chmod +x apktool`
* `mv apktool $PREFIX/bin/`
* `mv apktool.jar $PREFIX/bin/`
* `apktool --version`
